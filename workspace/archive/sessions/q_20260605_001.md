# 题目归档: q_20260605_001

## 题目
## 正误判断题

**题目：** 假定有数组 `int arr[5] = {1, 2, 3, 4, 5};`，声明指针 `int *p = arr;` 后，`p` 和 `arr` 在使用上完全等价，`sizeof(p)` 与 `sizeof(arr)` 的值相同。

请判断这句话的正误。

## 用户答案
错误

## 正确答案 + 解析
错误

题目说法错误。虽然 `p` 和 `arr` 在大部分场景可以互换（如 `p[i]` 和 `arr[i]` 都能访问数组元素），但 `sizeof` 的结果完全不同：`sizeof(arr)` 返回整个数组占用的字节数（5 × sizeof(int) = 20），而 `sizeof(p)` 返回指针变量本身的大小（8 或 4），与数组元素个数无关。数组名在 `sizeof` 下不发生退化，保留完整数组类型信息，而 `p` 只是普通指针变量。

## 讨论总结
### 错误根因
无

### 确认的知识点
- 数组名在 sizeof 下不发生指针退化，保留完整数组类型信息
- sizeof(arr) 返回整个数组大小，sizeof(p) 返回指针变量本身大小
- 除 sizeof 外，数组名在大多数表达式中会退化为指向首元素的指针
