# 题目归档: q_20260605_002

## 题目
关于 C++ 中的名字汇入机制，以下说法正确的是？

## 用户答案
C

## 正确答案 + 解析
C

正确答案是 C。using namespace 仅建立名字查找路径，名字本身仍然属于原来的名字空间，不是复制也不是移动实体。

## 讨论总结
### 错误根因
无（回答正确）
### 确认的知识点
- using namespace 是建立查找路径而非复制实体
- using N::x 和 using namespace N 范围不同
