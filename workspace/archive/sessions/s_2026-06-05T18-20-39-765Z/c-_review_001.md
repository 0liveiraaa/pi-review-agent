# 题目归档: c-_review_001

## 题目
判断以下说法是否正确：在 C++ 中，只有通过虚函数实现的动态绑定才算是多态，函数重载和模板不属于多态的范畴。

## 用户答案
错误

## 正确答案 + 解析
错误

❌ 说法错误。多态的核心定义是'相同的消息请求，在不同对象或不同参数条件下，执行不同的代码体'。函数重载和模板也满足'同名不同体'的条件，属于静态多态（编译期确定调用目标），而虚函数动态绑定属于动态多态（运行期参与决定）。不要把"多态"等同于"虚函数"。
