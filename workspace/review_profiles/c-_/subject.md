# C++_简短版

## 科目概述

C++ 面向对象程序设计课程，涵盖封装、继承、多态三大 OOP 特性。本资料包聚焦第 19 章"多态与虚机制"，从多态的基本概念出发，区划静态多态和动态多态，并深入讲解虚机制的原理与设计价值。

## 考试目标

- 能准确说出多态的定义（"相同消息请求产生不同执行体"），并区分静态多态和动态多态的触发时机
- 能识别函数重载和模板作为静态多态的典型例子
- 能解释 C++ 虚函数分派规则：目标对象按动态类型、参数按静态类型
- 能说明虚机制的核心设计价值：接口不变、实现可变、通过子类型化适应变化
- 能判断给定代码中虚函数调用的实际结果（多层继承、指针/引用场景）
- 能理解"面向接口编程"的设计思想及其与虚机制的关系

## 出题风格

- 题型偏好：选择题 40%、判断题 30%、简答题 30%
- 难度分布：S-R 10% / S-U 40% / M-U 35% / M-A 15%
- 出题特点：重辨析（静态多态 vs 动态多态、目标对象 vs 参数分派）、重设计理解（虚机制的设计价值而非虚表实现细节）、重代码分析（给定代码判断输出）
- 代码语言：cpp

## 资料说明

- 源文件数：2
- 覆盖范围：第 19 章 多态与虚机制（19.1 静态多态和动态多态、19.2 虚机制的作用）
- 资料类型：课程笔记（含代码示例、对比表格、考前速记）
