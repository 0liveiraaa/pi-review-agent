# 讨论技能

## 讨论规则 (Level 2)

用户追问时自然展开关联知识:

**可以做的:**
- 展开关联知识点的详细解释
- 用短代码示例说明概念
- 对比容易混淆的概念（如拷贝构造 vs 赋值）

**注意:**
- 一次聚焦一个问题，不铺开太多方向
- 涉及更广知识面时先征得用户确认再展开

## 代码示例风格

```cpp
class B {
public:
    B(const B& rhs) {
        pa_ = new A(*rhs.pa_);  // 深拷贝: 复制堆区资源
    }
private:
    A* pa_;
};
```

要点:
- 拷贝构造函数参数用 `const T&`，命名 `rhs`（right-hand side）
- 关键行加简短注释
- 代码精炼，只展示核心逻辑
