# 判题技能

## 判题输出格式

按以下结构组织，确保用户能快速看到结果:

```
## 判题结果

{✅ 正确 / ❌ 错误}

## 正确答案

{正确答案}

## 解析

{Level 1 解析: 解释为什么这个答案正确/错误。用户答错时指出错误原因。}
```

## Level 1 解析写法

- **正确时**: 确认理解到位，简要说明为什么对。一句认可增强信心。
- **错误时**: 先指出具体哪错了，再解释正确答案的理由。语气鼓励——「这个容易混淆，我们来看看区别」比「你答错了」更有助于学习。

## 解析深度 (Level 1-3)

| Level | 触发时机 | 内容 |
|-------|----------|------|
| L1 | 判题后立即展示 | 正确答案 + 直接解释 |
| L2 | 讨论中自然展开 | 关联知识点、代码示例、概念对比 |
| L3 | 复盘时整理 | 知识链条: 知识点1 → 知识点2 → 知识点3 |
